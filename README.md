# Consolidador-Aging
Aplicacion para consolidar agings
🌎 Consolidador de Aging - Aplicación Streamlit
Aplicación web para consolidar archivos aging excluyendo registros con PERSONA/CONSUMIDOR.
🚀 Instalación
Instalar dependencias:
```bash
pip install -r requirements.txt
```
Ejecutar la aplicación:
```bash
streamlit run app\_consolidador\_aging.py
```
La aplicación se abrirá automáticamente en tu navegador en: http://localhost:8501
📋 Características
✅ Subir múltiples archivos aging
✅ Consolidación automática multi-país
✅ Excluye registros con PERSONA/CONSUMIDOR
✅ Formato de fecha corta automático (dd/mm/yyyy)
✅ Estadísticas por país
✅ Vista previa de datos
✅ Descarga inmediata del resultado
🎯 Uso
Sube archivos: Click en "Browse files" y selecciona uno o más archivos .xlsx
Procesa: Click en "Consolidar archivos"
Descarga: Click en "Descargar archivo consolidado"
